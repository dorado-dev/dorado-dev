## About Me

I am a Software Developer focused on building modular, scalable, and highly maintainable systems. I work primarily within the Java (Spring Boot) and TypeScript (Angular) ecosystems, where my main priority is to decouple core business logic from framework-specific implementation details.

To achieve this, I rigorously apply Hexagonal Architecture and SOLID principles. I do not settle for code that simply "works"; I ensure every application is testable (JUnit/Mockito), secure by design (Stateless/JWT), and built to evolve without friction. I firmly believe that keeping the system design simple today is the most effective way to prevent technical debt tomorrow.

I view software development as the perfect balance between technical rigor and delivering real business value. I handle the complete application lifecycle—from relational data modeling and reactive UI state management to modern integrations involving Generative AI and RAG architectures. Ultimately, I write code keeping the next developer who will read it in mind.
